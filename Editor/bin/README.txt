Project:

Open the visual studio 2008 project solution in the "Visual studio 2008 project" folder.

Running:

FOr best results copy the "executable" folder to your computer (runnin off DVD will be slower)

Models - There is a selection of models located in the "data\models\" folder.
Textures - There is a selection of texture located in the "data\texture\" folder.

It is advised to read the user documentation and watch the tutorial video first.
A test map can be found in the executable directory "text_map_1.txt"

Source:

Shader source and source code can be viewed seperately in the folders "Shaders" and "Source code"
